# StudyApp - مشروع تطبيق دراسي محلي

**المهندس الكياني المستقل**

هذا المشروع هو تطبيق أندرويد دراسي متكامل، تم بناؤه وفقًا للمواصفات المطلوبة (محلي 100%، بدون إنترنت، تصميم Light Modern).

## الهيكل المعماري المعتمد

تم بناء التطبيق باستخدام **هندسة الطبقات (Layered Architecture)** لضمان فصل الاهتمامات وقابلية الصيانة، مع استخدام تقنيات أندرويد الأصلية لضمان الأداء والأمان.

| الطبقة | المكونات الرئيسية | التقنيات المستخدمة |
| :--- | :--- | :--- |
| **الواجهة (UI)** | `MainActivity`, `SubjectActivity`, `TaskActivity`, `LibraryActivity` | `XML`, `Material Design 3`, `CardView`, `RecyclerView` |
| **المنطق (Logic)** | `AppRepository` | Java |
| **البيانات (Data)** | `DatabaseHelper` | `SQLite` (للمواد، المهام، الملاحظات، الملفات) |
| **الخدمات (Services)** | `TaskWorker`, `FileUtils` | `WorkManager` (للتنبيهات), `FileProvider` (لأمان الملفات) |

## نظام الألوان المعتمد

تم اختيار نظام **Light Modern** (الخيار رقم 1):
*   **الخلفية:** أبيض ناصع ورمادي فاتح (`#F5F5F5`, `#FFFFFF`).
*   **اللون الأساسي:** بنفسجي ناعم (`#7C4DFF`).
*   **ألوان النصوص:** رمادي داكن ورمادي متوسط.

## كيفية الاستيراد في AIDE

1.  **فك الضغط:** قم بفك ضغط ملف `StudyApp.zip` الذي تم تسليمه لك.
2.  **الاستيراد:** افتح تطبيق AIDE على جهازك.
3.  **فتح المشروع:** اختر "Open existing project" (فتح مشروع موجود).
4.  **تحديد المسار:** انتقل إلى المجلد الذي فككت ضغطه (`StudyApp`) وحدده.
5.  **البناء والتشغيل:** سيقوم AIDE بتحميل المشروع. قد تحتاج إلى الضغط على زر "Run" (تشغيل) لبناء التطبيق وتثبيته على جهازك.

## الميزات المضمنة (حتى الآن)

*   **الصفحة الرئيسية:** هيكل واجهة المستخدم جاهز لعرض الإحصائيات وأقرب مهمة.
*   **المواد:** شاشة كاملة لإضافة وعرض المواد، مع مربع حوار مخصص لإضافة مادة جديدة.
*   **التذكيرات:** شاشة لعرض المهام والاختبارات القادمة، مع هيكل جاهز لجدولة التنبيهات عبر `WorkManager`.
*   **المكتبة:** شاشة لعرض المواد كمجلدات للملفات.
*   **قاعدة البيانات:** مخططات SQLite جاهزة لجميع الكيانات الأربعة (`subjects`, `notes`, `files`, `tasks`) مع دعم الـ `FOREIGN KEY` و `ON DELETE CASCADE` لضمان سلامة البيانات.

**ملاحظة:** تم بناء الهيكل الأساسي لجميع الميزات. يتطلب الأمر الآن استكمال منطق الواجهات المتبقية (مثل شاشة تفاصيل المادة، شاشة إضافة مهمة، شاشة الإعدادات) وربطها بالـ `AppRepository` المجهز.
